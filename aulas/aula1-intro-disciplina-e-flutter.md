## 4. O que é Flutter?

### 4.1 Flutter em poucas palavras

Flutter é um **framework de desenvolvimento de aplicações multiplataforma**, criado pelo Google, que permite desenvolver apps para:

* Android
* iOS
* Web
* Desktop

A partir de **uma única base de código**.

### 4.2 Flutter: maturidade e evolução recente

Nos últimos anos, o Flutter passou por uma **evolução significativa**, deixando de ser visto apenas como uma ferramenta para aplicações simples ou protótipos.

A integração cada vez mais profunda do Flutter com **serviços modernos de nuvem, APIs avançadas e soluções baseadas em Inteligência Artificial**, incluindo iniciativas recentes do próprio Google (como integrações com ferramentas de IA e plataformas de apoio ao desenvolvimento), reforça o amadurecimento do ecossistema.

Esse movimento evidencia que o Flutter **não é mais restrito a aplicações básicas de cadastro**, mas vem sendo utilizado em sistemas mais complexos, com maior volume de dados, regras de negócio mais elaboradas e integração com serviços inteligentes.

Esse cenário foi determinante para a decisão institucional de adotar o Flutter como tecnologia base da disciplina, considerando sua **evolução técnica**, **aderência ao mercado** e **potencial de longo prazo** para a formação do estudante.

> O Flutter evoluiu junto com o ecossistema de software moderno e hoje se posiciona como uma tecnologia adequada não apenas para aprender desenvolvimento mobile, mas para **construir aplicações reais, escaláveis e integradas a soluções contemporâneas**, incluindo Inteligência Artificial.

### 4.3 Modelo declarativo

Flutter utiliza um **modelo declarativo**, no qual:

* A interface é descrita como uma função do estado;
* Quando o estado muda, a interface é reconstruída;
* O foco é descrever *o que* deve ser exibido, não *como* atualizar a tela. É um pouco estranho de entender no início, mas fantástico depois que se aprende.

Este modelo será explorado gradualmente ao longo da disciplina.

## 5. Linguagem Dart

Dart é a linguagem utilizada pelo Flutter, moderna e projetada para produtividade e desempenho.
A partir de uma única base de código, é possível desenvolver aplicações multiplataforma, realizando ajustes específicos quando necessário para atender às particularidades de iOS e Android.

Características importantes:

* Tipagem forte (assim como C, TypeScript, Java e Go);
* Sintaxe moderna;
* Suporte nativo a programação assíncrona (tratamento de processamento assíncrono);
* Null Safety.

> 📌 Dart será introduzida de forma **progressiva**, sempre associada à prática.
