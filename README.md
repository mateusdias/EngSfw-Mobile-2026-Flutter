## 🎯 Sobre este repositório

Este é o **repositório base oficial da disciplina** de Tecnologia e Programação para Dispositivos Móveis (TPDM), com foco no desenvolvimento de aplicações móveis utilizando **Flutter**.

Aqui você encontrará:

- Notas de aula utilizadas pelo professor, em **Markdown**;
- Plano de ensino da disciplina;
- Cronograma organizado por blocos e entregáveis;
- Exemplos de código utilizados ao longo do semestre.

> ⚠️ **Atenção:**  
> Este repositório é **somente para consulta**.  
> As atividades práticas, exercícios e entregas **não devem ser realizadas aqui**.  
> Cada estudante possui um **repositório individual**, criado automaticamente por meio do **GitHub Classroom**.
